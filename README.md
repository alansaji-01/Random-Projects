Atomic Particle System with Hand Gesture Control
🎮 Live Demo & Repository Structure

✨ Features
Interactive Physics: Real-time particle simulation with electromagnetic forces

Hand Gesture Control: Control particles using your hand via webcam

Multiple Visualizations:

Electron orbits and trails

Bond visualization between atoms

Energy glow effects

Performance Monitoring: Real-time CPU/GPU/RAM/FPS monitoring

Molecular Structures: Pre-defined molecular formations

Camera/Mouse Support: Fallback to mouse control if camera not available

📋 Prerequisites
Python 3.8 or higher

Webcam (optional - for hand gesture control)

Modern CPU with SSE4.2 support
